
This repository contains a SystemVerilog UVM-based verification environment for the AMBA APB (Advanced Peripheral Bus) protocol. The project demonstrates industry-standard verification methodologies, including constrained-random verification, functional coverage, assertions, and reusable UVM components to verify APB read and write transactions.

The verification environment is designed to be modular, scalable, and reusable, making it suitable for learning UVM concepts as well as extending to more complex verification projects.
