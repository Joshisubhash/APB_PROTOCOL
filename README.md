# APB Protocol UVM Verification

This repository contains a **SystemVerilog UVM-based verification environment** for the **AMBA APB (Advanced Peripheral Bus)** protocol. The project showcases industry-standard verification methodologies, including **constrained-random verification, functional coverage, SystemVerilog Assertions (SVA), protocol checking, and reusable UVM components** to verify APB read and write transactions.

The verification environment is designed to be **modular, scalable, and reusable**, making it suitable for learning UVM concepts and serving as a foundation for more advanced verification projects.
